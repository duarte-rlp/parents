{% extends "global/Page.html" %} {% load otree static %} {% block scripts %} {% endblock %} {% block title %} Atividade 4: Lançar ou não lançar uma moeda {% endblock %} {% block content %} Nesta última atividade você decide se quer ou não apostar no lançamento de uma moeda.

Se você decidir lançar a moeda, você pode ganhar 18 reais se der COROA, ou perder uma certa quantia de dinheiro se der CARA. A possibilidade de sair cara ou coroa é sempre a mesma.

Se decidir não jogar a moeda, você não ganhará nem perderá.

Você verá 6 moedas diferentes. Sua tarefa é selecionar se deseja “Lançar” ou “NÃO lançar” cada uma das 6 moedas.

O que acontece se o computador selecionar esta atividade para calcular seu pagamento total?
O computador selecionará aleatoriamente uma das moedas. Se você decidir não lançar, você não receberá nenhum pagamento adicional. Se você decidir “Lançar”, o computador lançará a moeda e você ganhára 18 reais se der COROA ou perderá algum valor se der CARA.
